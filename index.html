Console-Based Coffee Shop System

The Console-Based Coffee Shop System is a text-based program that allows customers to view the coffee menu, place orders, manage their selections, and proceed to checkout through a command-line interface. It includes user authentication, menu display, order calculation, and basic inventory tracking, making it a simple yet functional model of a coffee shop ordering system.

Hotel Room Management System


The Hotel Room Management System is a web-based platform that enables staff to efficiently manage room reservations, check-ins, and check-outs. It features a real-time booking calendar that displays room availability, helping hotel administrators streamline operations while allowing guests to easily reserve rooms. This system improves accuracy, reduces double bookings, and ensures a smooth hotel management experience.
